MLDASH

uses http://socket.io.

It is capable of monitoring statistics from anything capable of producing discrete attributes. It has been successfully demonstrated with 2 use cases so far: Monitoring a server farm and the state of 36 battery cells in real time on a power flow controller

It summarises the information on a real-time dashboard available in a web interface.
