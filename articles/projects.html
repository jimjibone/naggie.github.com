DSPA

DSPA is an automated network based public announcement system.

Clients exist for google chrome and nodejs. Messages are automatically synthesised via the festival TTS engine and then played with lag compensation on any number of nodes at exactly the same time!

A location tree is used to allow messages to be sent to specific zones. This means a message can be sent to a specific room, house, or even city!

The google chrome extension is automatically built and deployed as an update to all existing clients.

MLDASH

uses http://socket.io.

It is capable of monitoring statistics from anything capable of producing discrete attributes. It has been successfully demonstrated with 2 use cases so far: Monitoring a server farm and the state of 36 battery cells in real time on a power flow controller

It summarises the information on a real-time dashboard available in a web interface.

Modular power flow controller

As my third year project for my MEng course at the university of Leicester, I developed a battery management system capable of monitoring and maintaining the health of a large Li-Ion based battery stack.

The system uses an optically isolated SPI bus to read the temperature and voltage of each battery module. The system is able to isolate any battery at any time, automatically, should the health of a cell depend on it.

The system could be used in circumstances where high availability and reliability is of foremost concern; for example, as part of a micro-grid (for renewable energy storage) where energy is required and available unpredictably.
